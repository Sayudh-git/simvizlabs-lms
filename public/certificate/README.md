# Certificate Images

This directory contains course completion certificates in PNG format.

## File Naming Convention
- `{courseId}.png` - Certificate for the specific course
- Example: `general.png`, `airbus_a320.png`, `boeing_737.png`

## Certificate Template
Each certificate should include:
- Course name
- Student name
- Completion date
- Institution logo
- Instructor signature
- Certificate number (optional)

## Generating Certificates
1. Open `generate_images.html` in a browser
2. Right-click on each certificate and "Save image as..." to download as PNG
3. Rename the files according to the naming convention
4. Place them in this directory

## Sample Certificates
The following sample certificates are available:
- `general.png` - Introduction to FMS
- `airbus_a320.png` - Airbus A320
- `boeing_737.png` - Boeing 737

## Adding New Certificates
To add a new certificate:
1. Create the certificate image (800x600px recommended)
2. Name it using the course ID (e.g., `new_course.png`)
3. Place it in this directory
4. The certificates page will automatically detect and display it
